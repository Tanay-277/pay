# PAY - Minimal Payment App

## Introduction

This project is a web application built using the MERN stack (MongoDB, Express.js, React, and Node.js). Its purpose is to provide users with a platform to manage their payments efficiently. 

## Features

- User Authentication
- User Profile Management
- Money Transfer Functionality

## Technologies Used

### Frontend

- React
- TypeScript
- ShadCN UI
- React Router
- Axios
- React Hook Form
- Tailwind CSS
- Zod

### Backend

- Node.js
- Express
- MongoDB
- JWT (JSON Web Tokens)
- bcrypt
- Zod

## Installation

### Prerequisites

- MongoDB
