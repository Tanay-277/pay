- Money Transfer: Dummy implementation for now, but the UI is fully functional and ready for integration.


## Tech Stack

- Frontend: ReactJS,TypeScript,React Hook Form,React Router
- UI: TailwindCSS, shadcn UI
- Backend: NodeJS,ExpressJS,JWT,Zod
- Database: MongoDB
